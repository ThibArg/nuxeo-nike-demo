nuxeo-nike-demo-mp
==================

#### Build
After cloning and assuming maven 3.2.1 is installed on your system:

First build nuxeo-nike-demo (see readme)

Then:
    ```
    cd /path/to/nuxeo-nike-demo-mp
    mvn install
    # The plug-in is in /target, named nuxeo-nike-demo-{version}.zip
    ```
